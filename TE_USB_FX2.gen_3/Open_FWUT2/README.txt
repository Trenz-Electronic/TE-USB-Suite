= Open FirmWare Upgrade Tool =
This software provided main functions to update EEPROM and Flash content on
TE modules based on Cypress USB-EZ FX2 chip.
This version compatible with TE Driver (Gen 3) API.


== Requirements ==
* Python >= 2.7
* Module Driver (Gen 3)

== Note ==
If you use 64-bit Windows, you should use 64-bit Python version.
